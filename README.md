## Installation

1. Navigate to the `custom_nodes` directory:
    ```sh
    cd custom_nodes
    ```
2. Clone the repository:
    ```sh
    git clone https://github.com/zohac/comfyui-shape-drawer.git
    ```
3. Restart ComfyUI.
